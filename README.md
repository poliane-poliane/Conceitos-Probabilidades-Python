# Probabilidade e Estatística com Python

Este projeto é um bloco de aprendizado focado na aplicação de conceitos fundamentais de **Probabilidade e Estatística** utilizando a linguagem **Python**.

## Sobre o Projeto
O objetivo deste repositório é documentar a transição da teoria estatística para a implementação prática via código. Aqui, utilizo bibliotecas como `pandas`, `numpy`, `matplotlib` e `seaborn` para analisar bases de dados simuladas e extrair insights estatísticos.

## Principais Conceitos Abordados
Atualmente, as anotações cobrem:
* **Análise Exploratória de Dados:** Utilização do método `.describe()` para entender métricas de tendência central e dispersão. Plotei alguns gráficos para aproveitar o espaço e indicar a importância de visualizar os dados em mais de um tipo de gráfico (aqui trouxe apenas Histograma e Boxplot).
* **Distribuições de Probabilidade:** Construção e visualização de distribuições para variáveis discretas.
* **Probabilidade Condicional:** Exercícios práticos sobre como informações adicionais restringem o espaço amostral e alteram probabilidades.
* **Visualização de Dados:** Criação de histogramas, boxplots e gráficos de barras para diagnosticar o comportamento dos dados.
